🎯 **React Interview Task – Question 1: Tabs Component** 🎯

This task is often asked in **React interviews** to evaluate a candidate’s understanding of component structure, state handling, and UI rendering. The goal is to **build a dynamic Tabs UI Component** using React and basic CSS.

---

## 📝 Problem Statement

> ✅ Create a **Tabs Component** using **React** that:
>
> - Renders a set of tab buttons dynamically from an array
> - Shows the content related to the currently active tab
> - Highlights the active tab button
> - Uses `useState` for state handling
> - Uses **clean CSS** for styling (no external libraries)

---

## 🔑 Key Features

- 🧠 **React State Management** using `useState`
- 🔄 **Dynamic Tabs** from array data
- 💡 **Conditional Styling** for active tab
- 🎨 **Custom CSS** for clean, responsive layout
- 🔁 **Scalable structure** — just update data to add more tabs

---

## 💻 Tech Stack

- ⚛️ React (Functional Components)
- 🧠 React Hooks (`useState`)
- 🎨 Plain CSS
- 🚀 Vite / CRA (Configurable)
